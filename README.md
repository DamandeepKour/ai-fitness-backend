# AI Fitness Backend

A scalable Node.js backend that powers an AI-driven fitness platform, generating personalized diet and workout plans using LLM APIs.

## 🚀 Tech Stack
- Node.js
- Express.js
- MySQL
- Redis
- OpenAI API

## 🧠 Architecture
- Controller → Service → Repository pattern
- Auto DB sync (no migrations)
- AI service layer with structured prompts
- Redis caching for performance

## ✨ Features
- AI-based diet & workout plan generation
- MySQL database with auto schema sync
- Redis caching for faster responses
- Clean modular architecture
- RESTful APIs
